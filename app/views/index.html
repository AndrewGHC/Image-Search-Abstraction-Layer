Image Abstraction Layer

Image Abstraction Layer - FCC Basejump

Enter a parameter in the format below to receive a response:

Receive an array of objects for an image search. Optional query 'offset' will affect pagination - ~/api/imagesearch/
Example: https://image-abstraction-layer.herokuapp.com/api/imagesearch/cats?offset=2

Retrieve all searches ordered by most recent first with ~/latest/imagesearch
Example: https://image-abstraction-layer.herokuapp.com/latest/imagesearch/
